# Código de Conduta

Esperamos que participantes mantenham um comportamento respeitoso e profissional. Condutas inaceitáveis incluem assédio, linguagem discriminatória, ameaças ou exposição de dados pessoais.

Se você for alvo ou testemunhar um caso, reporte para: **seu-email@dominio.gov.br** (contato responsável). As denúncias serão tratadas com confidencialidade.

Qualquer violação pode resultar em remoção do repositório ou comunicação às autoridades competentes.
